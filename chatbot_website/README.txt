Hello! Here is how to run the Insight AI chatbot:

1. Make sure you have Python installed on your computer.

2. Open your Terminal or Command Prompt in this project folder.

3. Install all the required Python libraries with this command:
   pip install -r requirements.txt

4. Download the English language model for the bot with this command:
   python -m spacy download en_core_web_sm

5. Run the web server with this command:
   python app.py

6. Open the `index.html` file in your web browser to start chatting!
